🔨 How to Build Your APK

⚡ Quick Start (5 minutes)

  1. Download the HandymanPro-SourceCode.zip above
  2. Extract the ZIP file to your computer
  3. Install Node.js if you don't have it: nodejs.org
  4. Open terminal in the handymanapp folder
  5. Run: npm install
  6. Build APK: npx eas build --platform android --profile preview --local

📱 Result: You'll get a fully working APK file ready to install!

💡 Alternative: If EAS doesn't work, try npx expo run:android - APK will be in android/app/build/outputs/apk/

📋 Installation Instructions

🎯 For Built APK Files:

  1. Build your APK using the source code above
  2. Enable "Install from Unknown Sources" on your Android device
  3. Transfer the APK to your phone
  4. Install the APK file
  5. Launch "Handyman Pro" from your app drawer
